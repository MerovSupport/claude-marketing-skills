# CONFIDENCE MAP — What You Can Trust

## CORE (10/10 Confidence — Use Without Hesitation)

- Weekly webinar until first million (Brunson rule, applied by Jean)
- Permission pivot script (quasi-verbatim from source)
- "If you teach they will buy" is a myth
- Start with 5 people, not 5000
- Progressive scaling: 50€/day → scale when profitable

## STRONG (8-9/10 — Reliable Principles)

- 5-step deck: Intro → 3 Secrets → Pivot → Offer → CTA+QA
- Offer formula: Promise + Product + Bonus + Guarantee + Authority + Urgency
- Multichannel show-up: email + SMS + WhatsApp + calendar + voice call
- 80% of success = presentation + offer (not traffic)
- Best ads don't look like ads
- Vulnerability creates trust
- Confused mind never buys — one offer per webinar
- Bonus value > product value
- <1000€ direct sale, >1000€ via call (French market)
- Skeleton structure, not word-for-word script

## MODERATE (5-7/10 — Use With Awareness)

- 40-50% show-up rate (may be peak, not average)
- 10-12% registration page conversion (unverified)
- ROAS 3x minimum (stated as "1€ in, 3€ out" but also mentions 10x)
- Email segmentation: absent / present-short / present-complete (from agency page, not Jean's words)
- Closing calls 20-30 minutes (stated once)
