# OBJECTION HANDLING — Réponses aux Objections

## PRINCIPE CLÉ

Les objections sont cassées PENDANT le contenu (les 3 secrets), PAS pendant le pitch. Les 3 secrets = les 3 plus grandes croyances limitantes à détruire.

**Méthode Jean :** Les histoires cassent les croyances en parlant à l'inconscient. La logique rationnelle ("ma méthode est la meilleure") active les défenses. Les histoires les contournent.

---

## OBJECTIONS ET RÉPONSES

### "Je n'ai pas le temps"

**Quand la casser :** Secret 1 ou 2

```
"La plupart des gens pensent qu'il faut y passer des heures chaque jour.
En réalité, le système que j'utilise nécessite [X heures/semaine].
[Exemple client] : il consacre 2 heures le dimanche soir à son webinaire.
Le reste est automatisé."
```

### "Je n'ai pas le budget"

**Quand la casser :** Contenu + stack + garantie

```
"Tu n'as pas besoin de budget pub pour commencer.
Fais des stories Instagram, invite tes contacts gratuitement.
Dès ta première vente, réinvestis une partie.
C'est comme ça que j'ai commencé : 15 inscrits, 1 vente à 997€.
150-200€ de pub suffisent pour commencer.
Tu CRÉES l'argent avec tes premières ventes."
```

### "Ça ne marchera pas dans ma niche"

**Quand la casser :** Données massives + diversité des cas

```
"J'ai fait plus de 2 000 webinaires dans plus de 30 niches différentes.
Lecture rapide, perte de poids, immobilier, e-commerce, guitare,
extensions de cils, filtres à eau, aquarelle, arbres, B2B, pâtisserie...
Ça marche dans toutes les niches.
La structure est la même. Seul le contenu change."
```

### "Je ne suis pas à l'aise en public"

```
"Au début, vous allez avoir 5-10 personnes. Pas la terre entière.
Et vous allez parler de VOTRE sujet, quelque chose que vous aimez.
Commencez à blanc, tout seul. Puis 3 personnes. Puis 10.
Moi-même, je suis derrière mon écran de base.
Vous n'êtes même pas obligé de montrer votre visage.
Vous pouvez faire des slides uniquement.
Le visage ajoute de la confiance, mais c'est pas obligatoire."
```

### "La publicité, c'est une arnaque"

```
"Commencer tout doucement, 50€ par jour.
90% du focus sur la créative — ce que tu DIS.
Tant que c'est rentable (1€ in, 3€ out), tu augmentes.
Sinon, tu ajustes.
Tout le monde a peur de la pub — c'est justement pour ça
que ceux qui la maîtrisent ont un avantage énorme."
```

### "Les webinaires, c'est dépassé"

```
"54 millions d'euros générés. 2 000+ webinaires. 30+ niches.
Les plus gros entrepreneurs français les utilisent
— Anthony Bourbon, Éric Larchevêque, tous.
Iman Gadzhi, 1 million d'inscrits sur son dernier challenge.
Le webinaire est le SEUL format qui donne 1h30
pour créer la confiance avec un prospect froid.
C'est le futur de la vente, pas le passé.
Le marché e-learning passe de 1 milliard à 3 milliards par jour."
```

### "Je ne veux pas avoir l'air d'un vendeur"

```
"Tu ne vends pas. Tu demandes la permission.
'Est-ce que c'est OK si je prends 10 minutes ?'
Les gens disent oui.
Ils n'ont pas l'impression de se faire vendre.
Et tu n'es pas obligé de vendre en direct.
Tu peux juste proposer un appel gratuit — un 'audit de candidature.'"
```

### "C'est trop cher"

**Stack visuel :**
```
Produit principal     → valeur 5 000€
Bonus 1              → valeur 2 000€
Bonus 2              → valeur 1 500€
Bonus 3              → valeur 1 000€
───────────────────────────────────
Valeur totale         = 9 500€
Prix aujourd'hui      = 997€

"Et avec la garantie [X], vous ne prenez aucun risque."
```

### "Le marché est saturé"

```
"On nous posait exactement cette question il y a 5 ans.
Le marché du e-learning passe de 1 milliard à 3 milliards par jour.
Le marché des produits digitaux : 180 milliards en 2020 → 340 milliards en 2025 → 800 milliards projeté en 2030.
Ce qui est saturé, c'est les clones qui copient-collent.
Ce qui marche, c'est un véhicule unique, un positionnement clair, une exécution propre."
```

### "Pas de produit / Je débute"

```
"Normal. Tu n'as pas besoin d'un produit parfait pour commencer.
Crée un MVP — le minimum viable product.
Les premiers modules de notre programme t'aident à trouver ta niche.
Maria ne savait même pas où cliquer.
Aujourd'hui elle fait 30-40K/mois en evergreen."
```

### "J'ai peur"

```
"Si j'ai peur, c'est le signal qu'il faut que j'y aille.
La peur n'est pas un mur. C'est un panneau indicateur.
Ton premier webinaire va être moyen. Je vous promets.
Fait est mieux que parfait."
```

### "Je peux le faire seul"

```
"Ne réinventez pas la roue.
Seul = 2-3 ans de tâtonnements.
Avec un système prouvé = accélérateur.
Compresser des décennies dans des jours."
```

### "Quelqu'un n'achète pas au premier contenu, il n'achètera jamais"

```
"Hormozi a montré que 90% de sa salle avait consommé
son contenu depuis 2-3 ans avant d'acheter.
Ne sois pas impatient.
Crée-toi une image de marque forte, fais de la qualité.
Tes clients satisfaits sont le meilleur marketing."
```

## MATRICE D'ANTICIPATION DANS LE DECK

| Moment du deck | Objection traitée |
|---------------|-------------------|
| Secret 1 | Objection principale (temps, argent, ou complexité) |
| Secret 2 | Objection secondaire (niche, compétences, légitimité) |
| Secret 3 | Objection tertiaire (risque, échec passé, marché saturé) |
| Offre — Véhicule unique | "En quoi c'est différent de ce que j'ai déjà essayé" |
| Offre — Garantie | Peur du risque financier |
| Offre — Témoignages | "Est-ce que ça marche vraiment ?" |
| Offre — Stack | "C'est trop cher" |
| Q&A | Toutes les objections restantes |

## COMMENT CASSER UNE CROYANCE (Méthode Jean)

```
1. IDENTIFIER la croyance (via sondage, appels, chat des webinaires précédents)

2. RACONTER UNE HISTOIRE qui contourne les défenses rationnelles
   → La tienne, ou celle d'un client dans la même situation
   → "Les histoires cassent les croyances en parlant à l'inconscient"

3. MONTRER LA RÉALITÉ (le shift)
   → Données, preuves, exemples concrets

4. DONNER UN TÉMOIGNAGE de quelqu'un qui avait la même croyance
   → "Il pensait exactement ça, et pourtant..."

5. NE PAS ARGUMENTER LOGIQUEMENT
   → "Si vous essayez de convaincre par A+B de manière rationnelle,
      le cerveau va être en mode 'Non, non, non.'
      Les histoires parlent à l'émotionnel, au cerveau inconscient."
```

## OBJECTIONS POST-WEBINAIRE (Email FAQ)

L'email C2 (J+2 pour les présents complets qui n'ont pas acheté) reprend les objections :

```
Objet : Les 5 questions qu'on me pose le plus

1. "Est-ce que ça marche dans [ma niche] ?"
   → Réponse + exemple client similaire

2. "Je n'ai pas le budget pour commencer"
   → Réponse + 150-200€ suffisent

3. "Et si ça ne marche pas pour moi ?"
   → Réponse + rappel garantie

4. "J'ai déjà essayé un webinaire et ça n'a pas marché"
   → Réponse + différence méthodologique (ton véhicule unique)

5. "Je ne suis pas expert, qui va m'écouter ?"
   → Réponse + exemples de débutants qui ont réussi
```
