# CAMPAIGN CHECKLIST — Webinaire Complet

## PRÉ-CAMPAGNE (J-30 à J-14)

### Recherche & Offre
- [ ] Sondage d'audience réalisé (rêves, objectifs, blocages, vocabulaire exact du client)
- [ ] 3 croyances limitantes principales identifiées
- [ ] Véhicule unique / nouvelle opportunité défini(e) — "En quoi ma méthode est DIFFÉRENTE ?"
- [ ] Offre : promesse + véhicule + bonus + garantie + urgence
- [ ] Prix défini (< 1 000€ = vente directe / > 1 000€ = appel)
- [ ] Bonus à haute valeur perçue créés (chaque bonus casse une objection)
- [ ] Garantie formulée
- [ ] Stack visuel préparé (valeur totale vs prix)

### Infrastructure
- [ ] Page d'inscription créée (objectif : 30%+ conversion depuis pub)
- [ ] Page de remerciement : confirmation + WhatsApp + calendrier + [tripwire optionnel]
- [ ] Plateforme webinaire configurée
- [ ] Séquence email automatisée (confirmation + J-3 + J-1 + J-0)
- [ ] SMS automatisé (J-0 -15 min)
- [ ] Appel vocal automatique (J-0 -5 min)
- [ ] Groupe WhatsApp ou communauté prêt
- [ ] Tracking (pixels, UTM, dashboard KPI)
- [ ] CRM configuré (si appels de vente)
- [ ] Lien de paiement ou booking prêt

## LE DECK (J-14 à J-7)

- [ ] Hook accrocheur (slide 1)
- [ ] Slide "pour qui / pas pour qui"
- [ ] Intro courte 5-10 min : histoire + crédibilité + 3 secrets annoncés
- [ ] Secret 1 : Histoire → Stratégie (pas tactique) → Témoignage → Transition
- [ ] Secret 2 : Histoire → Stratégie → Témoignage → Transition
- [ ] Secret 3 : Histoire → Stratégie → Témoignage → Transition
- [ ] Vérification : les 3 leviers de confiance sont activés (toi / produit / eux)
- [ ] Vérification : on reste en mode STRATÉGIE, pas tactique
- [ ] Pivot : demande de permission (non-agressif, quasi-verbatim de Jean)
- [ ] Offre : véhicule unique → produit → bonus → garantie → témoignages → stack → urgence
- [ ] CTA unique et clair (un seul lien, une seule action)
- [ ] Testé à blanc (seul ou avec famille/amis)
- [ ] Énergie vérifiée : le présentateur est dans un état d'énergie haute

## ACQUISITION (J-10 à J-0)

- [ ] Email à la liste existante
- [ ] Communauté notifiée (WhatsApp, Skool, Telegram)

- [ ] 3+ créatives différentes (ugly ad, face cam, témoignage, micro-trottoir...)
- [ ] Hook testé (plusieurs angles)
- [ ] Créatives alignées avec le message du webinaire (cohérence)
- [ ] Campagnes lancées J-7 à J-10
- [ ] Tracking ROAS en place

## RAPPELS MULTICANAUX (J-3 à J-0)

- [ ] J-3 : Email anticipation + valeur
- [ ] J-1 : Email + SMS rappel
- [ ] J-0 matin : Email rappel
- [ ] J-0 -15 min : SMS
- [ ] J-0 -5 min : Appel vocal auto
- [ ] J-0 : Message WhatsApp communauté

## JOUR DU WEBINAIRE

### Préparation
- [ ] Test technique (micro, caméra, slides, lien, chat)
- [ ] Lien de paiement ou booking testé et fonctionnel
- [ ] Eau, énergie haute, pas de distraction
- [ ] Chat activé

### Pendant le Live
- [ ] Introduction 5-10 min MAX (pas 55 min)
- [ ] Histoire + autorité (10 min)
- [ ] 3 secrets avec "aha moments" — mode STRATÉGIE
- [ ] Interactions chat régulières ("Dites OUI si...")
- [ ] Énergie montée progressivement
- [ ] Pivot : "Est-ce que c'est OK si..." (demander permission)
- [ ] Offre complète avec stack visuel
- [ ] CTA unique (lien direct ou booking appel)
- [ ] Q&A illimité — ne PAS couper court

### Après le Live
- [ ] KPIs notés immédiatement (inscrits, présents, conversions, CA)
- [ ] Chat analysé (quelles objections sont revenues ?)
- [ ] Notes d'amélioration pour le prochain webinaire

## POST-WEBINAIRE (J+1 à J+7)

- [ ] Segment Absents : email replay + SMS
- [ ] Segment Présents courts : email replay offre
- [ ] Segment Présents complets :
  - [ ] J+1 : offre + témoignage
  - [ ] J+2 : objections/FAQ
  - [ ] J+3 : urgence
  - [ ] J+4-5 : close cart
- [ ] Appels closers lancés (si high-ticket)
  - [ ] Calendriers ouverts 5-6 jours
  - [ ] Closers briefés sur l'offre et les objections
- [ ] Retargeting pub activé (ROI attendu x16-x25)
- [ ] Récupération paniers abandonnés

## DIAGNOSTIC RAPIDE

| Problème | Diagnostic | Solution |
|----------|-----------|---------|
| Peu d'inscrits | Pub ou promesse faible | Revoir headline page + créatives pub |
| CPL > 8€ | Créatives fatiguées | Refresh créatives, nouveaux hooks |
| Opt-in < 30% | Page d'inscription faible | Revoir headline, promesse, design |
| Inscrits mais peu présents | Multicanal insuffisant | Ajouter SMS, WhatsApp, calendrier, appel vocal |
| Présents mais partent vite | 10 premières minutes ratées | Revoir intro + "pourquoi rester" |
| Restent mais n'achètent pas | Croyances pas cassées ou offre faible | Revoir 3 secrets (sont-ils en mode stratégie ?) + offre |
| Achètent mais pas assez | Énergie faible ou offre pas assez empilée | Monter l'énergie + enrichir les bonus + stack |
| Appels qui ne closent pas | Prospects mal qualifiés ou closers faibles | Revoir qualification ou former closers |
| Follow-up faible | Séquence trop basique | Ajouter segmentation + retargeting + SMS |
| ROAS < 2.5x | Système global à optimiser | Identifier le maillon le plus faible et corriger |

## CHECKLIST SCALE (Quand le webinaire est validé)

- [ ] ROAS > 2.5x stable sur 3+ semaines
- [ ] Augmenter budget pub de 20-30% par semaine
- [ ] Recruter/former closers supplémentaires
- [ ] Ajouter de nouvelles créatives (10-15 à 500€/jour)
- [ ] Envisager l'evergreen quand Q&A sans surprise
- [ ] Planifier premier challenge (quand confiance dans le système)
