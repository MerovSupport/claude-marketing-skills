# CAMPAIGN CHECKLIST — Webinaire Complet

## PRÉ-CAMPAGNE (J-30 à J-14)

### Recherche & Offre
- [ ] Sondage d'audience réalisé (rêves, objectifs, blocages, vocabulaire)
- [ ] 3 croyances limitantes principales identifiées
- [ ] Offre : promesse + produit + bonus + garantie + autorité + urgence
- [ ] Prix défini (< 1 000€ = vente directe / > 1 000€ = appel)
- [ ] Bonus à haute valeur perçue créés
- [ ] Garantie formulée

### Infrastructure
- [ ] Page d'inscription créée (objectif : 10-12% conversion)
- [ ] Page de remerciement : confirmation + WhatsApp + calendrier + [tripwire]
- [ ] Plateforme webinaire configurée
- [ ] Séquence email automatisée
- [ ] SMS automatisé
- [ ] Appel vocal automatique (J-5 min)
- [ ] Tracking (pixels, Hyros, UTM)
- [ ] CRM configuré (si appels de vente)

## LE DECK (J-14 à J-7)

- [ ] Hook accrocheur (slide 1)
- [ ] Intro courte 5-10 min : histoire + crédibilité + 3 secrets annoncés
- [ ] Secret 1 : croyance limitante → réalité → shift → preuve
- [ ] Secret 2 : croyance limitante → réalité → shift → preuve
- [ ] Secret 3 : croyance limitante → réalité → shift → preuve
- [ ] Pivot : demande de permission (non-agressif)
- [ ] Offre : promesse → produit → bonus → garantie → témoignages → stack
- [ ] CTA unique et clair
- [ ] Testé à blanc (seul ou avec famille)

## ACQUISITION (J-10 à J-0)

### Organique
- [ ] Stories Instagram
- [ ] Posts réseaux sociaux
- [ ] Email à la liste existante
- [ ] Communauté notifiée

### Publicité
- [ ] Budget défini (50€/jour minimum)
- [ ] 3+ créatives différentes (ugly ad, face cam, témoignage...)
- [ ] Campagnes lancées J-7 à J-10
- [ ] Tracking ROAS en place

## RAPPELS (J-3 à J-0)

- [ ] J-3 : Email anticipation + valeur
- [ ] J-1 : Email + SMS rappel
- [ ] J-0 matin : Email rappel
- [ ] J-0 -15 min : SMS
- [ ] J-0 -5 min : Appel vocal auto
- [ ] J-0 : Message WhatsApp communauté

## JOUR DU WEBINAIRE

- [ ] Test technique (micro, caméra, slides, lien)
- [ ] Lien de paiement ou booking prêt
- [ ] Chat activé
- [ ] Intro 5-10 min (pas 55 min)
- [ ] 3 secrets avec "aha moments"
- [ ] Interactions chat
- [ ] Pivot : "Est-ce que c'est OK si..."
- [ ] Offre complète avec stack
- [ ] CTA unique
- [ ] Q&A illimité
- [ ] KPIs notés immédiatement

## POST-WEBINAIRE (J+1 à J+7)

- [ ] Segment Absents : email replay + SMS
- [ ] Segment Présents courts : email replay offre
- [ ] Segment Présents complets :
  - [ ] J+1 : offre + témoignage
  - [ ] J+2 : objections/FAQ
  - [ ] J+3 : urgence
  - [ ] J+4-5 : close cart
- [ ] Appels closers (si high-ticket)
- [ ] Récupération paniers abandonnés
