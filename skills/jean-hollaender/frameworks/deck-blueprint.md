# DECK BLUEPRINT — Architecture Détaillée

## STRUCTURE GLOBALE

```
BLOC 1 — HOOK & INTRO  (5-10 min)
         ↓
BLOC 2 — SECRET N°1   (15 min)
         ↓
BLOC 3 — SECRET N°2   (15 min)
         ↓
BLOC 4 — SECRET N°3   (15 min)
         ↓
BLOC 5 — PIVOT (2-3 min)
         ↓
BLOC 6 — OFFRE (10-15 min)
         ↓
BLOC 7 — CTA + Q&A (illimité)
```

**Durée totale :** 1h30 à 2h minimum. Jean et ses clients font souvent 3-5h (avec Q&A).
**Principe :** "Il n'y a pas de durée précise. La durée qui est la bonne, c'est celle qui vous permet de vendre."

---

## BLOC 1 — HOOK & INTRO (5-10 min)

### Objectif
- Capter l'attention immédiatement
- Donner envie de rester
- Établir la crédibilité
- Créer un lien humain

### Slides Type

| Slide | Contenu |
|-------|---------|
| **Slide 1 — Hook** | Titre accrocheur / promesse du webinaire. Même promesse que la page d'inscription |
| **Slide 2 — Qui suis-je** | Présentation rapide (5-10 min max !) Résultats, parcours, humilité |
| **Slide 3 — Ce que tu vas apprendre** | 3 bullets = les 3 secrets qu'on va couvrir |
| **Slide 4 — Pourquoi m'écouter** | Preuves sociales rapides |

### Template d'Intro

```
"Salut, je suis [nom]. J'ai [résultat crédibilité].
Aujourd'hui, pendant [durée], je vais vous montrer [promesse spécifique].
On va couvrir 3 choses :
1. [Secret 1 - curiosité]
2. [Secret 2 - curiosité]
3. [Secret 3 - curiosité]
Restez jusqu'à la fin parce que [raison de rester]."
```

### Règles
- **Vulnérabilité > Perfection** — exposer ses erreurs crée la confiance
- **Pas de script rigide** — ne pas lire mot à mot
- **Intro COURTE** — 5-10 min max. "Jean a dit à Francis : prends 10 min. Il a pris 55 min. Ce n'est pas ce qu'il fallait."

## BLOCS 2, 3, 4 — LES 3 GRANDS SECRETS (45 min total)

### Objectif de Chaque Secret
1. Apporter de la **vraie valeur** (pas un teasing vide)
2. **Casser une croyance limitante** spécifique
3. Créer un **"aha moment"** → shift mental
4. Amener le prospect à se dire **"c'est possible pour moi"**

### Structure de Chaque Secret

```
Secret N° [1/2/3] : [TITRE ACCROCHEUR] — 15 minutes chacun
│
├── 1. HISTOIRE (ouverture)
│   → Raconter une histoire (la tienne ou celle d'un client)
│   → "Les gens adorent les histoires et c'est ce qui casse les croyances"
│   → L'histoire introduit la croyance limitante à casser
│
├── 2. STRATÉGIE (contenu)
│   → Ta méthode / ton approche sur ce point
│   → Sois pédagogue et simple
│   → Le but : prouver que TON VÉHICULE (ta méthode) est le meilleur
│
├── 3. TÉMOIGNAGE (closing)
│   → Termine par un témoignage client
│   → "Le témoignage amplifie ton marketing"
│   → Preuve que ça marche pour d'autres
│
└── 4. TRANSITION
    "Maintenant qu'on a vu [secret N], passons à..."
```

### Le Concept du "Véhicule" (Source: interview Antoine Blanco)

Le but des 3 secrets n'est pas juste d'enseigner — c'est de convaincre que TA MÉTHODE (ton "véhicule") est la meilleure pour obtenir le résultat voulu.

"Les gens ont tout entendu, ils voient plein de choses à droite à gauche. Le but de ton webinaire c'est convaincre que ton véhicule, ta méthode, c'est ce qui va leur permettre d'aller chercher le résultat."

**Repackaging :** Tu peux changer le marketing de ton véhicule (nouveau nom, nouvel angle) même si la méthode reste la même.

### Comment Choisir les 3 Secrets

Les 3 secrets = les 3 plus grandes objections/croyances limitantes du prospect :

| Croyance limitante typique | Secret correspondant |
|---------------------------|---------------------|
| "Je n'ai pas le temps" | Secret sur l'efficacité/les raccourcis |
| "Je n'ai pas l'argent" | Secret sur le ROI/le coût de l'inaction |
| "Ça ne marchera pas pour moi" | Secret avec preuve par les cas similaires |
| "C'est trop compliqué" | Secret qui simplifie le processus |
| "J'ai déjà essayé" | Secret sur pourquoi les autres méthodes échouent |

### L'Équilibre Critique
Assez de valeur pour créer confiance, PAS assez pour que le prospect pense qu'il n'a plus besoin de ton offre.

### Ce que ce N'EST PAS
- ❌ Un cours complet (les gens partent avec "trop à appliquer")
- ❌ Un teasing vide (les gens se sentent arnaqués)
- ❌ Un PowerPoint corporate (c'est une conversation)

## BLOC 5 — LE PIVOT (2-3 min)

### Script du Pivot (Quasi-Verbatim de Jean)

```
"Voilà, on arrive à la fin de cette conférence.

Est-ce que ça vous a plu ? [Attendre les réactions dans le chat]

Super, merci ! J'ai adoré passer ce moment avec vous.

Écoutez, est-ce que c'est OK si je prends les 10 prochaines minutes
juste pour vous présenter quelque chose sur lequel je travaille
depuis [X mois/années] et qui me tient vraiment à cœur ?

Dites-moi si c'est OK pour vous.

Et pour ceux qui veulent y aller, allez-y, je ne vous retiens pas."
```

### Mécanismes Psychologiques

| Mécanisme | Effet |
|-----------|-------|
| "Est-ce que ça vous a plu ?" | Micro-engagement positif |
| Demander l'autorisation | Le prospect dit OUI = engagement psychologique |
| "Ceux qui veulent partir, allez-y" | Liberté = pas de pression = confiance |
| "Depuis X mois" | Montre l'investissement personnel |
| "Qui me tient à cœur" | Émotion, authenticité |

**Résultat :** Les gens ne se sentent PAS vendus. Ils se sentent invités.

## BLOC 6 — L'OFFRE (10-15 min)

### Séquence de Présentation

```
1. PROMESSE
   "Voici ce que [nom de l'offre] va vous permettre d'accomplir..."

2. PRODUIT PRINCIPAL
   "Voici exactement ce que vous allez recevoir..."

3. BONUS (un par un)
   "En plus, vous allez aussi recevoir..."
   - Bonus 1 : [Nom] (valeur X€) — simplifie [aspect]
   - Bonus 2 : [Nom] (valeur X€) — accélère [aspect]
   - Bonus 3 : [Nom] (valeur X€) — garantit [aspect]

4. GARANTIE
   "Et pour que vous n'ayez absolument aucun risque..."

5. PREUVES / TÉMOIGNAGES
   "Voici ce que [prénom] a accompli..."
   → 2-3 témoignages

6. STACK VISUEL (slide récapitulative)
   Produit principal          → valeur X€
   Bonus 1                   → valeur X€
   Bonus 2                   → valeur X€
   Bonus 3                   → valeur X€
   ────────────────────────────────
   Valeur totale              = XXXX€
   Prix aujourd'hui           = YYY€

7. URGENCE / EXCLUSIVITÉ
   "Cette offre est disponible uniquement pour les participants"
   ou "Les X premières places seulement"
   ou "Le prix remonte dans 48h"
```

### Règle des Bonus (Leçon Hormozi)
"Les bonus doivent avoir tellement de valeur que les gens sont prêts à acheter l'offre JUSTE pour un bonus."

## BLOC 7 — CTA + Q&A (Illimité)

### CTA

| Type d'offre | CTA |
|-------------|-----|
| < 1 000€ | Lien d'achat direct — "Cliquez sur le lien pour accéder maintenant" |
| > 1 000€ | Calendrier booking — "Réservez votre appel avec mon équipe" |

**UNE SEULE ACTION.** Un seul lien. Un seul bouton.

### CTA Vente Directe (< 1 000€)

```
"Si tout ce que je viens de vous montrer résonne avec vous,
voici comment aller plus loin.
Cliquez sur le lien qui s'affiche.
Vous allez pouvoir accéder à [offre] avec tous les bonus.
Et rappelez-vous : [garantie] donc vous ne prenez aucun risque."
```

### CTA Appel (> 1 000€)

```
"Pour ceux qui sont intéressés, on a un nombre limité de places
pour un appel avec mon équipe.
C'est un appel de 20-30 minutes, 100% gratuit,
où on va voir ensemble si [offre] est fait pour vous.
Cliquez sur le lien pour réserver votre créneau."
```

### Q&A
- Rester **aussi longtemps que nécessaire**
- "Hormozi a fait 7 heures. Nous, on fait 3, 4, 5 heures."
- Chaque question = une objection adressée en public
- Chaque réponse rassure tous ceux qui avaient la même question
