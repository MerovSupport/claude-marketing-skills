# DECK BLUEPRINT — Architecture Détaillée du Webinaire

## STRUCTURE GLOBALE (6 PHASES)

```
PHASE 1 — INTRODUCTION     (5-10 min)
         ↓
PHASE 2 — HISTOIRE & AUTORITÉ (10 min)
         ↓
PHASE 3 — SECRET N°1       (15-20 min)
         ↓
PHASE 4 — SECRET N°2       (15-20 min)
         ↓
PHASE 5 — SECRET N°3       (15-20 min)
         ↓
PHASE 6 — PIVOT            (2-3 min)
         ↓
PHASE 7 — OFFRE            (15-30 min)
         ↓
PHASE 8 — CTA + Q&A        (illimité — 1h à 5h)
```

**Durée totale :** 1h30 à 2h minimum (sans Q&A). Avec Q&A : 3-5h courant, 7h possible.

**Principe :** "Il n'y a pas de durée précise. La durée qui est la bonne, c'est celle qui vous permet de vendre."

---

## LES 3 LEVIERS DE CONFIANCE — Grille de Vérification

Chaque phase du webinaire doit activer au moins un des 3 leviers :

| Phase | Confiance en TOI | Confiance en ton PRODUIT | Confiance en EUX |
|-------|:----------------:|:------------------------:|:-----------------:|
| Introduction | ✅ (crédibilité) | | |
| Histoire | ✅ (identification) | | ✅ (c'est possible) |
| Secret 1 | ✅ (expertise) | ✅ (véhicule) | ✅ (cas similaires) |
| Secret 2 | ✅ (expertise) | ✅ (véhicule) | ✅ (cas similaires) |
| Secret 3 | ✅ (expertise) | ✅ (véhicule) | ✅ (cas similaires) |
| Pivot | ✅ (authenticité) | | |
| Offre | | ✅ (stack, bonus) | ✅ (garantie) |
| Q&A | ✅ (patience, écoute) | ✅ (réponses) | ✅ (rassurance) |

## PHASE 1 — INTRODUCTION (5-10 min)

### Objectif
- Capter l'attention immédiatement
- VENDRE le fait de rester 90 minutes
- Donner envie de ne pas décrocher

### Slides Type

| Slide | Contenu |
|-------|---------|
| **Slide 1 — Hook** | Titre accrocheur / promesse du webinaire |
| **Slide 2 — Pour qui / Pas pour qui** | Attirer les bons, repousser les mauvais |
| **Slide 3 — Ce qu'on va voir** | 3 bullets = les 3 secrets (curiosité) |
| **Slide 4 — Consignes** | Fermer les onglets, prendre de l'eau, être focus |

### Template d'Intro

```
"Salut, je suis [nom]. Aujourd'hui, pendant les 90 prochaines minutes,
je vais vous montrer [promesse spécifique].

On va couvrir 3 choses :
1. [Secret 1 - curiosité]
2. [Secret 2 - curiosité]
3. [Secret 3 - curiosité]

Cette conférence est pour vous si [critères].
Elle n'est PAS pour vous si [anti-critères].

Fermez vos onglets, prenez de l'eau, et restez jusqu'à la fin
parce que [raison de rester]."
```

### Règles
- **Intro COURTE** — 5-10 min max. "Jean a dit à Francis : prends 10 min. Il a pris 55 min. Ce n'est pas ce qu'il fallait."
- **Énergie haute** — donner le ton dès le début
- **Pas de script rigide** — ne pas lire mot à mot

---

## PHASE 2 — HISTOIRE & AUTORITÉ (10 min)

### Objectif
- Créer du lien émotionnel
- Montrer que tu es passé par les mêmes douleurs
- Établir l'autorité sans arrogance

### Structure

```
1. TON PASSÉ — La situation "avant" (douleurs, obstacles, croyances limitantes)
   → "J'étais exactement là où vous êtes"
   → Le prospect doit s'IDENTIFIER

2. LE DÉCLIC — Ce qui a changé (découverte, mentor, méthode)
   → Le moment pivot

3. LA TRANSFORMATION — Où tu en es aujourd'hui
   → Résultats, mais avec humilité

4. L'AUTORITÉ — Pourquoi m'écouter ?
   → Preuves sociales rapides (chiffres, témoignages, trophées)
   → "On ne crée pas l'autorité en la revendiquant, on la crée en la démontrant"
```

### Règles
- **Vulnérabilité > Perfection** — exposer ses erreurs crée la confiance
- **Pas de script rigide** — squelette + conversation naturelle
- **Pas d'arrogance** — l'autorité vient de la preuve, pas de la revendication
- **Maximum 10 min** — résister à la tentation de trop parler de soi

## PHASES 3, 4, 5 — LES 3 SECRETS (40-60 min total)

### Objectif de Chaque Secret
1. **Casser UNE croyance limitante** spécifique
2. **Prouver que TON VÉHICULE** (ta méthode) est le meilleur
3. Créer un **"aha moment"** → shift mental
4. Amener le prospect à se dire **"c'est possible pour moi"**
5. **Monter l'énergie** — enthousiasme, patate, momentum

### Structure de Chaque Secret

```
Secret N° [1/2/3] : [TITRE ACCROCHEUR] — 15-20 minutes
│
├── 1. HISTOIRE (ouverture)
│   → Raconter une histoire (la tienne ou celle d'un client)
│   → "Les histoires cassent les croyances en parlant à l'inconscient"
│   → L'histoire introduit la croyance limitante à casser
│   → Contrairement à la logique rationnelle, l'histoire contourne les défenses
│
├── 2. STRATÉGIE (contenu) ⚠️ PAS TACTIQUE
│   → Ta méthode / ton approche sur ce point
│   → VISION AÉRIENNE : les grandes étapes, pas le détail
│   → "C'est comme un oiseau qui voit le chemin — pas Google Maps"
│   → La tactique = c'est ce que les gens PAIENT dans ta formation
│   → Le but : prouver que ton véhicule est NOUVEAU et DIFFÉRENT
│
├── 3. TÉMOIGNAGE (closing)
│   → Termine par un témoignage client
│   → "Le témoignage amplifie ton marketing"
│   → Preuve que ça marche pour d'AUTRES (confiance en eux)
│
└── 4. TRANSITION
    "Maintenant qu'on a vu [secret N], passons à..."
```

### Le Concept du "Véhicule" / Nouvelle Opportunité

> "Les gens s'ils sont sur votre webinaire, c'est qu'ils ont déjà testé des choses. Montrer que votre méthode est DIFFÉRENTE — c'est un véhicule nouveau qui change tout."

Les 3 secrets doivent prouver que :
1. La manière traditionnelle ne marche PAS (pourquoi ils ont échoué avant)
2. Il existe une NOUVELLE approche (ton véhicule)
3. Cette approche marche pour des gens comme eux (preuve)

### Comment Choisir les 3 Secrets

Les 3 secrets = les 3 plus grandes croyances limitantes / objections :

| Croyance limitante typique | Secret correspondant |
|---------------------------|---------------------|
| "Je n'ai pas le temps" | Secret sur l'efficacité / le système |
| "Je n'ai pas l'argent" | Secret sur le ROI / le coût de l'inaction |
| "Ça ne marchera pas pour moi" | Secret avec preuves de cas similaires |
| "C'est trop compliqué" | Secret qui simplifie le processus |
| "J'ai déjà essayé" | Secret sur pourquoi les AUTRES méthodes échouent (→ ton véhicule est différent) |
| "Le marché est saturé" | Secret avec données marché + angle unique |

### La Règle Stratégie vs Tactique

> "Ne rentrez JAMAIS dans la tactique ou la technique et restez dans la stratégie."

| ✅ STRATÉGIE (webinaire) | ❌ TACTIQUE (formation) |
|--------------------------|------------------------|
| Les 5 grandes étapes | Le détail de chaque étape |
| Le "quoi" et le "pourquoi" | Le "comment" précis |
| La vision d'ensemble | Le pas-à-pas |
| "Voici le chemin" | "Tournez à droite, puis à gauche" |

### L'Énergie

> "Ce qui fait que les gens passent à l'action, c'est qu'ils sont dans ce mood, cette émotion où ils se disent 'OK, je suis capable, c'est possible, je peux le faire.'"

- Envoyer de l'énergie, de l'enthousiasme, de la patate
- Le présentateur doit être dans un état d'énergie haute
- L'énergie est CONTAGIEUSE — si tu es plat, ils seront plats
- Interactions chat régulières ("Dites-moi OUI dans le chat !")

### Ce que ce N'EST PAS
- ❌ Un cours complet (les gens partent avec "trop à appliquer")
- ❌ Un teasing vide (les gens se sentent arnaqués)
- ❌ Un PowerPoint corporate (c'est une conversation)
- ❌ De la tactique détaillée (c'est ce qu'ils paient dans ta formation)

## PHASE 6 — LE PIVOT (2-3 min)

### Script du Pivot (Quasi-Verbatim de Jean)

```
"Voilà, on arrive à la fin de cette conférence.

Est-ce que ça vous a plu ? [Attendre les réactions dans le chat]
Dites-moi si ça vous a plu !

Super, merci ! J'ai adoré passer ce moment avec vous.

Est-ce que vous pensez que c'est possible pour vous ?
Est-ce que vous pensez que vous êtes capable ?
[Laisser le chat répondre — succession de OUI]

Écoutez, est-ce que c'est OK si je prends les 10 prochaines minutes
juste pour vous présenter un énorme projet sur lequel je travaille
depuis [X mois/années] et qui me tient vraiment à cœur ?

C'est extraordinaire pour les personnes qui veulent aller plus loin.

Dites-moi si c'est OK pour vous.

Et pour ceux qui veulent y aller, allez-y, je ne vous retiens pas."
```

### Mécanismes Psychologiques

| Mécanisme | Effet |
|-----------|-------|
| "Est-ce que ça vous a plu ?" | Micro-engagement positif |
| "Est-ce que c'est possible pour vous ?" | Active la confiance en eux |
| Succession de OUI | Momentum d'engagement |
| Demander l'autorisation | Ils disent OUI = engagement psychologique |
| "Ceux qui veulent partir, allez-y" | Liberté = pas de pression = confiance |
| "Depuis X mois" | Investissement personnel visible |
| "Qui me tient à cœur" | Émotion, authenticité |

> "Les gens ne se sentent PAS vendus. Ils se sentent invités."

## PHASE 7 — L'OFFRE (15-30 min)

### Séquence de Présentation

```
1. PROMESSE
   "Voici ce que [nom de l'offre] va vous permettre d'accomplir..."

2. VÉHICULE UNIQUE
   "Ce qui rend [nom] différent de tout ce que vous avez essayé..."

3. PRODUIT PRINCIPAL
   "Voici exactement ce que vous allez recevoir..."

4. BONUS (un par un — chacun casse une objection)
   - Bonus 1 : [Nom] (valeur X€) — élimine [peur/obstacle]
   - Bonus 2 : [Nom] (valeur X€) — accélère [aspect]
   - Bonus 3 : [Nom] (valeur X€) — garantit [aspect]

5. GARANTIE
   "Et pour que vous n'ayez absolument aucun risque..."

6. PREUVES / TÉMOIGNAGES
   "Voici ce que [prénom] a accompli..."
   → 2-3 témoignages

7. STACK VISUEL (slide récapitulative)
   Produit principal          → valeur X€
   Bonus 1                   → valeur X€
   Bonus 2                   → valeur X€
   Bonus 3                   → valeur X€
   ────────────────────────────────────
   Valeur totale              = XXXX€
   Prix aujourd'hui           = YYY€

8. URGENCE / EXCLUSIVITÉ
   "Cette offre est disponible uniquement pour les participants"
   ou "Les X premières places"
   ou "Le prix remonte dans 48h"
```

### Règle des Bonus (Hormozi)
"Les bonus doivent avoir tellement de valeur que les gens sont prêts à acheter l'offre JUSTE pour un bonus."

## PHASE 8 — CTA + Q&A (Illimité)

### CTA

| Type d'offre | CTA |
|-------------|-----|
| < 1 000€ | Lien d'achat direct |
| > 1 000€ | "Audit de candidature" (appel 20-30 min) |

**UNE SEULE ACTION.** Un seul lien. Un seul bouton.

### CTA Appel (> 1 000€)

```
"Pour ceux qui sont intéressés, on a un nombre limité de places
pour un audit de candidature avec mon équipe.
C'est un appel de 20-30 minutes, 100% gratuit,
où on va voir ensemble si [offre] est fait pour vous.
Cliquez sur le lien pour réserver votre créneau.
Candidatures ouvertes jusqu'à [deadline]."
```

**"Audit de candidature"** = inverse la dynamique. C'est EUX qui candidatent. Pas toi qui vends.

### Q&A
- Rester **aussi longtemps que nécessaire** — les gens DOUBLENT leurs ventes pendant le Q&A
- "Plus les gens passent de temps avec vous, plus le capital confiance monte, plus ils achètent. C'est mathématique."
- Chaque question = une objection adressée en public

### Après le Q&A — La Boucle d'Amélioration

1. **Analyser le chat** immédiatement après le webinaire
2. **Identifier les objections** qui sont revenues fréquemment
3. **Si une objection revient souvent** → elle n'a pas été cassée dans les 3 secrets → l'intégrer dans le prochain deck
4. **Objectif à long terme** : un webinaire tellement bon que personne n'a plus de questions au Q&A → PRÊT POUR L'EVERGREEN
