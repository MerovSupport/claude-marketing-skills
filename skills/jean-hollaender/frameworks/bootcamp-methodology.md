Jean appelle sa méthodologie le "PROFIT Method". Détails enseignés dans le cours Liberty Webi.

## LES 3 PHASES DE CROISSANCE

```
Phase 1 — CRÉER
→ Créer ton webinaire, atteindre 10K€/mois
→ 60% des élèves commencent ici
→ Durée moyenne : ~6 semaines pour premier 10K

Phase 2 — STABILISER
→ Structurer, systématiser
→ Équipe, process, CRM

Phase 3 — EXPAND
→ Scale via challenges, evergreen, événements
→ Objectif 100K+/mois
```

---

## LE MOTEUR BUSINESS EN 4 ÉTAPES

```
1. ÊTRE VISIBLE      → Trafic (pubs + organique)
2. GÉNÉRER DES LEADS → Page inscription (convertir visiteurs en inscrits)
3. CRÉER LA CONFIANCE → Le webinaire (la structure vend pour toi)
4. VENDRE            → 5-10% des présents convertissent
```

"C'est la structure qui vend pour vous."

## MATHS WEBINAIRE DÉTAILLÉES

### Modèle Standard
```
1,000 inscrits × ~5€ CPL = 5,000€ budget pub
× 30-40% show-up = 300-400 présents
× 5-10% conversion = 15-40 ventes
× 1,000€ prix moyen = 15,000€ - 40,000€
+ follow-up (1/3 à 1/2 des ventes en plus)
= 20,000€ - 60,000€ total

ROI = x4 à x12
```

### Budget de Démarrage
- **150-200€** suffisent pour les premières ventes
- Réinvestir les profits
- "Pas besoin de 5,000€ pour commencer — tu CRÉES cet argent avec tes premières ventes"

### Temps Moyen
- Premier 10K€ : ~6 semaines
- Avec méthode appliquée A-Z

## STRUCTURE DU CHALLENGE (Comment Jean organise le bootcamp)

### Format : 5 Soirées Live

| Jour | Contenu | Durée réelle |
|------|---------|-------------|
| Jour 1 | Introduction + système de vente + guests | 7h (annoncé 2-3h) |
| Jour 2 | Offre + présentation + pitch | 7h |
| Jour 3 | Acquisition + trafic + scaling | 6-7h |
| Jour 4 | Optimisation + témoignages + guests | 6-7h |
| Jour 5 | Synthèse + dernière offre + close | 6-7h |
| Bonus | Best-of + panel alumni + pitch final | 4.5h |

### Règles du Challenge
- **Ne PAS pitcher Jour 1** — construire la confiance d'abord
- Pitcher à partir de Jour 2
- **Témoignages live (3-6 élèves) CHAQUE soir**
- Les sessions durent 2-3x plus longtemps que prévu (générosité = confiance)
- Q&A illimité chaque soir
- "Promo 2026" comme identité de cohort dans le chat

### CTA du Challenge : "Audit de Candidature"
Appel gratuit reframé en "évaluation de candidature" :
- Inverse la dynamique de pouvoir (ils vous SÉLECTIONNENT)
- "L'audit est offert" (anciennement 500€)
- "Ça vous engage à rien"
- Calendrier de booking dans le chat
- Hard deadline : "Candidatures ouvertes jusqu'à minuit"
- Post-booking : questionnaire + vidéo obligatoire + cas études + Trustpilot

## PITCH STRUCTURE EN 17 ÉTAPES (Comment Jean Vend)

C'est sa méthodologie EN ACTION — un meta-exemple de sa propre méthode.

```
1.  Vidéo social proof (montage célébrations élèves)
2.  Re-établir l'autorité (50M€, 2000 webinaires, trophées)
3.  Origin stories personnelles (vulnérabilité, combat)
4.  Exercice du POURQUOI (participation, ancrage émotionnel)
5.  Phase éducation (one-to-one vs many vs infinite)
6.  Démo cas d'étude (Mohamed: petit appart → L'Olympia)
7.  Maths concrètes (5€/lead, 350 présents, 25 ventes, 25K€)
8.  Preempt objections ("pas besoin de budget", "pas besoin de tech")
9.  Stacking de preuves (vidéos témoignages, niches diverses)
10. Reveal de l'offre (8 modules + 12 accélérateurs + billets + garantie)
11. Future pacing ("qu'est-ce que vous ressentiriez si...")
12. CTA : Audit de candidature (lien, places limitées)
13. Panel alumni live (10+ personnes réelles)
14. Push motivation final (monologue philosophique Jody)
15. Soft close repeat (lien, rappel minuit)
16. Q&A (niches, prix, timeline)
17. Teaser prochaine session
```

### Rôles Jean vs Jody
- **Jean** = vente technique/structurelle (maths, process, méthode)
- **Jody** = vente émotionnelle/philosophique (pourquoi, peur, action)
