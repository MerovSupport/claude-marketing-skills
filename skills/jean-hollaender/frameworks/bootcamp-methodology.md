## QUAND FAIRE UN CHALLENGE

### Arbre de Décision

```
Tu as un webinaire live rentable ?
├── NON → Ne fais PAS de challenge. Retourne au webinaire live.
│         "Ça demande beaucoup d'énergie et quand vous vous trompez,
│          vous perdez beaucoup d'argent, de temps et d'énergie."
│
└── OUI → Tu peux envisager un challenge.
          ├── Budget pub disponible ? → OUI → Lancer un challenge
          └── NON → Continuer le webinaire hebdo, accumuler du budget
```

### Webinaire vs Challenge vs VSL

| Format | Quand | ROI | Avantage | Inconvénient |
|--------|-------|-----|----------|-------------|
| **Webinaire** | TOUJOURS en premier | x3-x5 | Simple, rapide, testable | Volume limité par session |
| **Challenge** | Quand webinaire validé | x10-x15 | Confiance maximale, volume massif | Extrêmement lourd, risqué |
| **VSL** | Très rare | x3-x4 max | Automatisé, scalable | Requiert copywriting + closers exceptionnels |

> "Le challenge c'est en fait un webinaire sur 5 jours d'affilée. C'est comme un accordéon — au lieu de faire ça sur une soirée, on fait sur cinq soirées."

## LA MÉTHODE PROFITS — DÉCODÉE

```
P = PUBLICITÉ      → Trafic / Acquisition
R = REGISTRATION    → Page d'inscription / Opt-in (30%+)
O = ONBOARDING      → Accueil pré-événement (thank-you page + séquences)
F = FRÉQUENTATION   → Taux de présence (multicanal)
I = INTERACTION     → Le show live (présentation)
T = TRANSACTION     → La vente (offre + pitch)
S = SUIVI           → Follow-up post-événement
```

Chaque lettre = un levier. Améliorer chaque levier de 10-15% = doubler le business total.

## STRUCTURE DU CHALLENGE (5 Soirées)

### Format

| Jour | Contenu | Durée réelle |
|------|---------|-------------|
| **Jour 1** | Introduction + système de vente + guests | 6-7h (annoncé 2-3h) |
| **Jour 2** | Offre + présentation + pitch | 6-7h |
| **Jour 3** | Acquisition + trafic + scaling | 6-7h |
| **Jour 4** | Optimisation + témoignages + guests | 6-7h |
| **Jour 5** | Synthèse + dernière offre + close | 6-7h |
| **Bonus** | Best-of + panel alumni + pitch final | 4-5h |

### Règles du Challenge

1. **Ne PAS pitcher Jour 1** — construire la confiance d'abord
2. **Pitcher à partir de Jour 2**
3. **Témoignages live (3-6 élèves) CHAQUE soir** — personnes réelles, pas des acteurs
4. **Les sessions durent 2-3× plus longtemps que prévu** — la générosité = la confiance
5. **Q&A illimité chaque soir**
6. **Identité de cohorte** dans le chat (ex: "Promo 2026")
7. **Créer un mouvement**, pas juste un funnel
8. **Pression stratégique** : ajouter de NOUVEAUX bonus chaque jour, changer le pitch chaque jour

### La Pression Stratégique (≠ Agressivité)

> "Ne pas vendre agressivement tous les jours. Ajouter de nouveaux bonus quotidiennement. Changer le pitch quotidiennement."

| Jour | Stratégie de pression |
|------|----------------------|
| Jour 1 | PAS de pitch — construction de confiance |
| Jour 2 | Premier pitch + offre de base |
| Jour 3 | Offre + nouveau bonus exclusif Jour 3 |
| Jour 4 | Offre + nouveau bonus exclusif Jour 4 + témoignages |
| Jour 5 | Offre complète + urgence finale + deadline |

## CTA DU CHALLENGE : "AUDIT DE CANDIDATURE"

Appel gratuit reframé en "évaluation de candidature" :

- **Inverse la dynamique** — ils candidatent auprès de TOI
- "L'audit est offert" (anciennement 500€)
- "Ça vous engage à rien"
- Calendrier de booking dans le chat
- **Hard deadline** : "Candidatures ouvertes jusqu'à minuit"
- Post-booking : questionnaire + vidéo obligatoire + cas d'études + Trustpilot

## LES 5 PATTERNS DES LANCEMENTS À 1M€+

1. **Créer un MOUVEMENT** — pas juste un funnel. Un message central, une expérience, une identité de cohorte. Les gens se sentent partie d'un mouvement.

2. **Pression STRATÉGIQUE** — pas d'agressivité. Ajouter de nouveaux bonus quotidiennement. Changer le pitch quotidiennement. Créer un momentum croissant.

3. **Offre basée sur la TRANSFORMATION** — pas sur le prix. La transformation promise justifie tout prix.

4. **Follow-up MASSIF** — 30% des ventes viennent APRÈS l'événement.
   - Témoignages émotionnels pour les émotionnels
   - FAQ détaillée pour les rationnels
   - Urgence réelle (deadline)
   - Retargeting (ROI x16-x25)

5. **ÉQUIPE dédiée** — media buyer, équipe créative, tech/funnel, closers, pédagogie/delivery.
