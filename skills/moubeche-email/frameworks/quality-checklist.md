# Quality Control Checklist

## Pre-Send Checklist (Every Email)

### Technical
- [ ] SPF/DKIM/DMARC configured on sending domain
- [ ] Sending from dedicated outreach domain (not main domain)
- [ ] Email warm-up completed (3-5 weeks)
- [ ] Volume: < 100 emails/day per sender
- [ ] No attachments (links only, Google Drive preferred)
- [ ] Simple signature (no complex HTML)
- [ ] Opt-out/unsubscribe present
- [ ] Custom tracking domain active

### Copy Quality
- [ ] Subject line: 1-8 words, casual, no spam triggers
- [ ] Opening: about the prospect, demonstrates research
- [ ] Word count: 50-125 (cold) / appropriate for type
- [ ] Framework applied (PAS, PIN, Proof-First, etc.)
- [ ] Single CTA, matched to trust level, under 7 words
- [ ] "You/your" count > "I/we/our" count (at least 2:1)
- [ ] Reading level: 3rd-5th grade
- [ ] No spam trigger words
- [ ] No exclamation marks (max 1)
- [ ] No emojis (reduce reply rate by 13%)
- [ ] Every sentence earns its place (no filler)

### Personalization
- [ ] Tiramisu intro: specific research about the prospect
- [ ] Context: why NOW, why THEM
- [ ] 3-4 personalization variables used appropriately
- [ ] Reviewed each personalized message individually [Moubeche principle]

### Sequence Logic
- [ ] 4-9 follow-ups planned
- [ ] Each follow-up adds NEW value
- [ ] Timing: 2→4→4→5+ day gaps
- [ ] Follow-ups in same thread (no new subject lines)
- [ ] Breakup email included
- [ ] Total sequence: 10-25 days

### Strategic Alignment
- [ ] Audience awareness level identified
- [ ] Email goal matches funnel stage
- [ ] CTA matches trust level
- [ ] Consistent with landing page / next step
- [ ] Part of a coherent sequence (not random)

## Pre-Campaign Checklist

- [ ] Target list verified (bounce rate < 2%)
- [ ] Segments defined by pain point
- [ ] Minimum 100 prospects per A/B test variation
- [ ] Baseline metrics established
- [ ] A/B test designed (testing ONE variable)
- [ ] Multichannel steps mapped (email + LinkedIn + phone)
- [ ] Reply handling process defined (who responds, when, how)
- [ ] CRM integration active
- [ ] Campaign labeled and organized

---

## Pre-Sequence Checklist

- [ ] Full sequence mapped out (all emails drafted)
- [ ] Logical progression across emails
- [ ] Value escalation: each email builds on the last
- [ ] No "just checking in" follow-ups
- [ ] Breakup email is genuinely useful (referral ask, not guilt)
- [ ] Multichannel touchpoints scheduled

## Post-Campaign Review

### Weekly Review
| Metric | Target | Actual | Status |
|--------|--------|--------|--------|
| Open rate | > 50% | | |
| Reply rate | > 8% | | |
| Positive reply rate | > 5% | | |
| Bounce rate | < 2% | | |
| Unsubscribe rate | < 5% | | |
| Spam reports | < 0.1% | | |
| Meetings booked | [target] | | |

### Diagnostic Actions
| If This | Then This |
|---------|-----------|
| Open rate < 40% | Check deliverability → fix SPF/DKIM/DMARC → review subject lines |
| Open rate 40-50% | A/B test subject lines |
| Reply rate < 3% | Review copy → personalization → CTA → targeting |
| Reply rate 3-8% | A/B test body copy and CTA format |
| High bounces (> 5%) | Clean list, verify emails before next campaign |
| High unsubscribes | Review targeting and frequency |
| Spam reports > 0.1% | Reduce volume, increase spacing, review content |

### Monthly Strategy Review
- [ ] Which sequences performed best? Why?
- [ ] Which segments converted highest?
- [ ] What messaging angles resonated?
- [ ] What should be killed or paused?
- [ ] What new tests should run next month?
- [ ] Are there new segments to target?
- [ ] Is content supporting outbound? [Moubeche principle]

## Skill Output Quality Check

Before delivering ANY output from this skill, verify:

1. **Is it specific?** No vague advice. Every recommendation has a concrete action.
2. **Is it commercially sharp?** Tied to revenue, conversion, or retention metrics.
3. **Is it actionable?** The user can implement it today, not "eventually."
4. **Is it honest about sources?** [Moubeche principle] vs [Inferred extension] labeled.
5. **Does it avoid generic marketing language?** No "engage your audience" or "add more value."
6. **Does it match the user's context?** Adapted to their specific business, audience, offer.
7. **Would Guillaume Moubeche nod at this?** Does it reflect his reply-first, trust-building, execution-focused philosophy?
